# 相关代码实践

本目录用于存放论文相关的代码实现和实验。

## 目录组织

每个实践项目建立独立子目录，包含：

```
project_name/
├── README.md          # 项目说明
├── requirements.txt   # 依赖管理
├── src/              # 源代码
├── notebooks/        # 实验记录
└── data/            # 数据集
```
